# Retail Sales Data Analysis

## Overview
This project analyzes a retail dataset (~10,000 rows) to identify trends in sales and profitability.

## Tools Used
- SQL (MySQL)
- Microsoft Excel

## Files Included
- superstore.csv: dataset used for analysis
- superstore.sql: SQL queries used to analyze the data
- Retail_Sales_Analysis_Report.pdf: summary of findings and insights

## Key Tasks
- Cleaned and organized data for analysis
- Wrote SQL queries using filtering, grouping, and aggregation
- Analyzed sales, profit, and discount trends

## Key Insights
- Higher discounts often led to lower profit margins
- Certain regions generated higher revenue but lower profit
- Product categories showed different performance trends

## Summary
This project demonstrates foundational data analysis skills and the ability to extract insights from structured data.
